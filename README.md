# 🏆 GitHub Badge Quest 🚀

This repository is dedicated to exploring GitHub's gamification features and unlocking profile badges. It serves as a testing ground for Pull Requests, Actions, and Community Discussions.

---

## 🎖️ Badge Progress Tracker

| Badge | Level | Strategy | Status |
| :--- | :---: | :--- | :---: |
| **Pull Shark** | 1 | Opened 2 merged Pull Requests. | 🔄 |
| **Quickdraw** | — | Closed an issue/PR within 5 min. | ⚡ |
| **YOLO** | — | Merged a PR without a review. | ✅ |
| **Galaxy Brain** | — | Answered a question in Discussions. | 🧠 |
| **Pair Extraordinaire** | 1 | Co-authored a commit with a friend. | 🤝 |
| **Arctic Code Vault** | — | Contributed code to a hidden vault. | ❄️ |

## 🛠️ Triggering the Unlocks

### 🦈 Pull Shark & YOLO
To unlock these, I follow this workflow:
1. Create a new branch: `git checkout -b feature/update.`
2. Make a small change to this README.
3. Push and open a **Pull Request**.
4. Merge it immediately (triggers **YOLO**).
5. Repeat to level up **Pull Shark**.
